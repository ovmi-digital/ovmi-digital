# ovmi.tech

Marketing website for Ovmi — a software studio based in Europe building products across web, mobile, AI, and blockchain.

Built with [Astro](https://astro.build/) and [Tailwind CSS v4](https://tailwindcss.com/).

## Getting Started

**Prerequisites:** [Bun](https://bun.sh/)

```sh
bun install
bun dev
```

Open [http://localhost:4321](http://localhost:4321).

## Scripts

| Command       | Description                       |
| ------------- | --------------------------------- |
| `bun dev`     | Start dev server                  |
| `bun build`   | Build for production into `dist/` |
| `bun preview` | Preview the production build      |

## Project Structure

```
src/
├── components/    # Astro components (Hero, About, Services, etc.)
├── layouts/       # Base layout
├── pages/         # Routes (single-page site)
└── styles/        # Global CSS
public/
├── fonts/
├── images/
└── favicon.svg
```
